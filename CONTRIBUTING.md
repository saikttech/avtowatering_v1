# Как внести вклад

## Стиль кода
- Используйте комментарии на русском языке
- Функции должны быть не длиннее 50 строк
- Все новые функции добавляйте в прототипы в начале файла
- Следуйте существующему стилю именования

## Перед PR
- Протестируйте код на реальном устройстве
- Обновите версию в шапке файла
- Обновите README при изменении функционала
- Проверьте совместимость со старыми EEPROM-конфигами
